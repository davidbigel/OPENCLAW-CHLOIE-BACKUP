# research

Standard output root for Research Categorization Agent deliverables.

- [entities/](entities/) - one company/entity categorization file per entity.
- [batches/](batches/) - multi-company or task-level batch outputs.
- [sources/](sources/) - source notes, captured references, and evidence logs when useful.
- [templates/](templates/) - reusable Markdown schemas and output templates.

Default behavior: save completed research outputs into the relevant subfolder unless David asks for a different location.
